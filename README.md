webaltakiller
=============
Tool for removing webalta malware

Как собирать?
=============
Программа собиралась с помощью Rad Studio Delphi XE.
Кроме самого Delphi потребеутся Overbyte ICS v6(и выше) - http://www.overbyte.be/.
